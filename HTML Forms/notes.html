HTML Forms

represents a document section containing interactive controls for submitting information

Accept-charset

space separated character encodings the server accepts

Autocapitalize

Controls whether inputted text is automatically capitalized and if so in what matter

none or off = do not automatically capitalize any text

sentences or on = automatically capitalize the first character of each sentences

words = automatically capitalize the first character of each word

character = automatically capitalize every character

Auto complete

Whether input elements can be default have their values automatically completed by the browser

off = browser cant automatically complete sentences

on = browser can automatically complete entries

Name

the name of the form.

The value must not be the empty string and must be unique among the forms elements in the forms collection that it is in

Rel

Define the relationship between the current document or part of the document and the linked resource

external = if the form needs to be send to an external source

ATTRIBUTES

Action

The url that processes the form submission

Action attribute cant be EMPTY

Enctype

Specifies the encoded type that is used when submitting the form data to the server

it defines how the data should be formatted and transmitted incase of simple data, file uploads or other type of data

Method

HTTP Method to submit the form

Novalidate

Disable the validation of the data

Form shouldnt be validated when submitted

Target

Where to display the response after submitting the form

Current page, current tab, iframe or window

_self = current tab

_blank = new page

_parent = loads into the parent browser (if not parent then acts as _self)

_top = loads into top-level browsing context
