CSS Properties and function notation that use color

Direct color value usage

  • Color = sets the color of the text
  • background-color = sets the background color of the element
  • border-color = sets the color of the border
  • outline-color = sets the outline color
  • caret-color = sets the color of the cursor when hovering the text
  • text-decoration-color = sets the color of the text decorators
  • column-rule-color = sets the color of the line drawn between columns in a multi-column layout

Color value in a list or as an argument

  • text-shadow = adds shadow to text
  • border = a shorthand property that uses the value of the border-color CSS property in the list of values
  • linear-gardient() = creates an image consisting of a progressive transition between two or more colors along a straight line.
