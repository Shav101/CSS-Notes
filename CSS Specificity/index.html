CSS SPECIFICITY

Specificity is the algorithm used by the browser to determine the CSS declaration that is the most relevant to an element, which in return, determines the property value to apply to the element.

  • Inline style = 1000
  • Type selector = 001
  • Pseudo-element selector = 001
  • Class selector = 010
  • Pseudo-class selector = 010
  • Attribute selector = 010
  • ID selector = 100
  • Universal selector = 000
  • Compound selector = x y z
  • CSS combined = x y z
