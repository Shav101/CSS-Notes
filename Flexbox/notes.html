Major Axis?

Its the axis that the items get to placed according to

Horizontal Axis (Default of a flexbox axis)

Vertical Axis

we use "Displey: Flex" to move things in our horizontal axis

but they are usually stacked closely to each other

JUSTIFY-CONTENT

we will be using "justify-content" to "move content along the major axis"

left to right

space-between | space-around | space-evenly | flex-start | flex-end | center

FLEX-WRAP

Wraps your content (pretty much moves the item to new line)

Anything that overflows you can wrap it

wrap | wrap-reverse | initial | inherit

Gap

Adds a space between elements along major axis

size unit

ALIGN-ITEMS

Move content along MINOR axis

from top to bottom

space-between | space-around | space-evenly | flex-start | flex-end | center

FLEX-DIRECTION

changing the direction of the major axis

defines which axis is the major axis

row | row-reverse | column | column-reverse | initial | inherit

FLEX-FLOW

short hand for flex direction flex-wrap

row wrap

DISPLAY

how we set the display of a container

we need "displey:flex" in order to turn something into a flexbox

ORDER

The order CSS property sets the order to lay out an item in a flex or grid container.

Items in a container are sorted by ascending order value and then by their source code order.

Items not given an explicit order value are assigned the default value of 0.

FLEX

flex:

The flex CSS shorthand property sets how a flex item will grow or shrink to fit the space available in its flex container.
