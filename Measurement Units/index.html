MEASUREMENT UNITS IN CSS

Absolute measurement units

They are a units that are dont depend on any base value

they are not calculated directly from another value

PX = Pixels *1px = 1/96th of 1in*

Relative measurement units

Always have a base value they get calculated from

The value is either provided by the element they are used for or by a parent element or by a root element of the html document

em = relative to the font-size of the element

rem = relative to font-size of the root element

vw = relative to 1% of the width of the viewpoint

vh = relative to 1% of the height of the viewpoint

% = relative to the parent element
